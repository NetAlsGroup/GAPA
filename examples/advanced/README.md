# Advanced Examples

These examples are intentionally not the first stop for new users.

Recommended order:

1. `../quickstart_local.py`
2. `../custom_algorithm_minimal.py`
3. advanced remote / multi-node examples in this folder

Files:

- `remote_single_server.py`
  - thin wrapper around `run_sixdst.py` for remote `s` / `sm` / `m`
- `mnm_multi_node.py`
  - thin wrapper around `run_sixdst.py` for heterogeneous MNM mode
- repo-root advanced scripts kept for compatibility:
  - `../resource_scheduler.py`
  - `../run_analysis_queue.py`
  - `../run_lock_keepalive.py`
  - `../run_report_export.py`
  - `../run_trends.py`
  - `../run_perf_baseline.py`
